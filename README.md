# Procesamiento Batch con PySpark

## Descripción
Este proyecto implementa un análisis batch sobre un archivo CSV con datos del valor del dólar.  
Se realizan operaciones de limpieza y análisis exploratorio (EDA), incluyendo:
- Estadísticas generales.
- Promedio del valor.
- Máximo y mínimo.
- Conteo de registros.
- Visualización de los primeros registros.

## Requisitos
- Python 3.8 o superior
- Apache Spark 3.5.x
- Java 8 o superior
- Librerías adicionales:
  ```bash
  pip install pyspark
